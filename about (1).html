ProvocareaThe challenge

AI real, fără server și fără costReal AI, no server, no cost

Un evaluator de CV credibil are nevoie de inteligență reală, nu de reguli fixe. Dar portofoliul trăiește pe GitHub Pages — fără backend, fără chei sigure. Cum livrezi AI autentic fără a expune o cheie API sau a genera costuri pentru oricine deschide link-ul? A credible CV evaluator needs real intelligence, not fixed rules. But the portfolio lives on GitHub Pages — no backend, no safe place for keys. How do you ship genuine AI without exposing an API key or billing anyone who opens the link?

Decizii tehniceTechnical decisions

Problemă → Decizie → Rezultat

01

Cost zero, oriundeZero cost, anywhere

ProblemăProblem

Un apel API expus pe Pages = cheie publică și costuri reale pentru oricine.An API call exposed on Pages = a public key and real costs for anyone.

DecizieDecision

Detectare automată a mediului: API real în Claude.ai, motor local euristic oriunde altundeva. Nicio cheie în cod.Automatic environment detection: real API inside Claude.ai, a local heuristic engine everywhere else. No key in the code.

RezultatResult

Demo complet funcțional pe Pages — fără card, fără cheie, fără factură.A fully working demo on Pages — no card, no key, no invoice.

02

PDF citit în browserPDF read in-browser

ProblemăProblem

Extragerea textului dintr-un PDF cere de obicei un backend.Extracting PDF text usually requires a backend.

DecizieDecision

pdf.js încărcat la cerere de la CDN, extrage textul direct în browser.pdf.js loaded on demand from a CDN, extracting text directly in the browser.

RezultatResult

PDF-uri reale, analizate local, gratuit. Cele scanate sunt detectate și utilizatorul e ghidat.Real PDFs analyzed locally, for free. Scanned ones are detected and the user is guided.

03

Comutare RO/EN fără regenerareRO/EN switching without regeneration

ProblemăProblem

Întrebările generate o singură dată rămâneau în limba inițială.Questions generated once stayed in their original language.

DecizieDecision

Fiecare întrebare e un descriptor independent de limbă; textul se derivă din limba curentă.Each question is a language-independent descriptor; the text is derived from the current language.

RezultatResult

Toggle-ul re-traduce instant întrebări, istoric și feedback, fără apeluri în plus.The toggle instantly re-translates questions, history and feedback, with no extra calls.

04

Istoric persistent cross-environmentCross-environment persistent history

ProblemăProblem

localStorage e blocat în artifacts; window.storage nu există pe Pages.localStorage is blocked in artifacts; window.storage doesn't exist on Pages.

DecizieDecision

Un strat de stocare care alege automat window.storage → localStorage → memorie.A storage layer that auto-picks window.storage → localStorage → in-memory.

RezultatResult

Istoricul persistă în orice mediu, fără server.History persists in any environment, with no server.
